# Power BI Notes 📊

Detailed, structured notes on Power BI — covering everything from data import to building interactive dashboards.

## Contents

- **Introduction** – What is Power BI, its components, and workflow
- **Importing Data** – Data sources and connectivity modes (Import, DirectQuery, Live)
- **Power Query** – Data transformation (ETL), Applied Steps, M language basics
- **Data Modeling** – Relationships, star schema, date tables
- **DAX** – Measures vs calculated columns, context, and key function categories with examples
- **Visualizations** – Chart types, when to use them, and interactivity (slicers, drill-through, bookmarks)
- **Publishing & Sharing** – Power BI Service, workspaces, RLS, gateways
- **Extras** – Commonly missed topics like incremental refresh, dataflows, and performance tuning
